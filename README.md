LecteurComics
===============

A web app to read comics book (CBR/CBZ/CBT)

##Keyboard Shortcuts

UP: zoom in
DOWN: zoom out
LEFT: previous picture
RIGHT: next picture

## Included code

LecteurComics includes:

* [Bitjs](https://code.google.com/p/bitjs)

Reference:
* [kthoom] (https://code.google.com/p/kthoom)
* [Openberg] (https://github.com/Yoric/openberg)
